# Basic Egg Timer
this is a simple timer that starts at 5 minutes by default with the ability to add time using an "Add Minute" button 

made using Godot 4, runs on windows

(Using Godot for a simple timer program is overkill, I am just trying to explore what I can do with it :) )

## Download/Install
Downlad the .zip file from the main release, extract it and run the .exe
## Features
- Start timer button
- Add minute button
- Pause/Play buttons
- RESET button
- Stop alarm button
  - (appears when alarm goes off)
- Close timer button
  - (the biggest flower on the top of the timer's window)
 - Draggable transparent window
